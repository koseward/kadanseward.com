Technical Debt

Overview

Technical debt is an metric in computer engineering that describes the amount of overhead in a project as the result of project fast-forwarding. Or, in other words, it is "the debt incurred through the speeding up of software project development which results in a number of deficiencies ending up in high maintenance overheads.[1]
