Repository

Overview

A repository, in terms of software development, is a central file storage location, similar to a folder. A repository is different however, in that it allows for branching and version control. Branching allows for different features being implemented seperate from the main project before being integrated into the main system.

Relation to Sofware Architecture

Repositories are often referred to when discussing the modularity of a software architecture. For example, a monolithic architecture has low modularity, due to its single repository, while microservices have many repositories for each service, and therefore have higher modularity.
